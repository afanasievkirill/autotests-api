## Предварительные настройки.

Подготовить виртуальное окружение Python.

```
$ python -m venv venv
```

Активировать виртуальное окружение Python.

```
$ .\\venv\Scripts\activate
```

Установить зависимости.

```
$ pip install -r requirements.txt
```
